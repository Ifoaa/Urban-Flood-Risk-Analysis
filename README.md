Urban Flood Risk Assessment of Port Harcourt

Project Overview

This project investigates flood susceptibility in Port Harcourt, Rivers State, Nigeria, using GIS and climate data. The analysis integrates terrain characteristics and rainfall patterns to identify areas potentially vulnerable to flooding.

Objectives
* Analyze the influence of elevation on flood risk.
* Identify low-lying and flood-prone areas..
* Generate flood susceptibility maps using GIS techniques.
* Explore spatial patterns that may support urban planning and flood mitigation.


Datasets
* Digital Elevation Model (DEM)
* Derived Slope Raster
* Rainfall Data (NASA Climate Data)
* Rivers State LGA Boundaries


Tools Used
* QGIS
* Python
* Pandas
* GeoPandas
* Raster Analysis
* Spatial Analysis


Outputs
* DEM Map
* Slope Map
* Hillshade Map
* Rainfall Analysis
* Flood Susceptibility Map


Limitations

A flood prediction model was explored but not included in the final analysis due to limited historical flood-event data, which resulted in overfitting and unreliable predictions. Future work will incorporate larger datasets and validated flood records.
